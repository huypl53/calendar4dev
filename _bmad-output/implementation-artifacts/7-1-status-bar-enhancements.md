# Story 7.1: Status Bar Enhancements

Status: ready-for-dev

## Story

As a user,
I want the status bar to show contextual information,
so that I have ambient awareness of my calendar state.

## Acceptance Criteria

1. Status bar shows keyboard shortcut hint (Cmd+K for palette, ? for help)
2. Status bar shows current view mode
3. Sync indicator with timestamp of last data fetch

## Tasks / Subtasks

- [ ] Task 1: Add shortcut hints to status bar
- [ ] Task 2: Show current view label
- [ ] Task 3: Sync status with last fetch time

## What NOT to Build

- **No WebSocket real-time sync** — deferred to post-MVP
- **No cross-tab sync** — deferred
